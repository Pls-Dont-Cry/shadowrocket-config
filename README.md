# Shadowrocket Config

## Как использовать
Открыть [Shadowrocket](https://www.shadowrocketdownload.com) И добавить конфигурацию из URL-адреса: 
`https://raw.githubusercontent.com/Pls-Dont-Cry/shadowrocket-config/master/shadowrocket.conf`

## Предопределенные правила

 - https://github.com/blackmatrix7/ios_rule_script
